# webfrontend
KIKI商城前端
基于Vue2.0开发 基本使用elment-ui组件搭建页面
感谢hyh大佬的帮助

## Node.js命令

### 项目构建
```
npm install
```

### 开发调试
```
npm run serve
```

### 开发发布
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
